# Sales Data Analysis Project

This project involves analyzing sales data with a focus on key metrics such as the number of customers, purchase amounts and quantities by category, average basket value, satisfaction scores, and purchase frequency. The goal is to gain insights into customer behavior and sales performance to support data-driven business decisions.

Key analysis areas include:
- Customer count and segmentation
- Purchase amounts and quantities categorized by product groups
- Calculation of average basket value
- Evaluation of customer satisfaction scores
- Analysis of purchase frequency and patterns

Tools used in this project include Excel for data management and Power BI for creating interactive dashboards and visualizations.
